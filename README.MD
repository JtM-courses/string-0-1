## Install

```bash
$ sudo apt install docker docker.io docker-compose
$ sudo usermod -aG docker $USER
$ newgrp docker 
```

[Documentation](https://docs.docker.com/install/linux/linux-postinstall/)

## Run

```bash
$ docker-compose up -d
```

## Stop

```bash
$ docker-compose down
```
